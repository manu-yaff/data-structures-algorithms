# Big O time complexities of Data Structures and Algorithms

| Algorithm/Structure | Insertion | Removal | Searching | Access |
| ---- | --------- | ------- | --------- | ------ |
| Objects | O(1) | O(1) | O(n) | O(1) |
| Arrays | O(n) | O(n) | O(n) | O(1) |

# Big O of Array Operations

| Operation | Time Complexity |
| ---- | --------- |
| push | O(1) |
| pop | O(1) |
| shift | O(n) |
| unshift | O(n) |
| concat | O(n) |
| slice | O(n) |
| splice | O(n) |
| sort | O(n * log n) |
| forEach/map/filter/reduce... | O(n) |
